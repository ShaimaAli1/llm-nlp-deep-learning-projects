# LLM, NLP & Deep Learning Projects

A collection of applied deep learning and NLP projects spanning classical CNNs, embeddings, transformer fine-tuning, and retrieval-augmented generation (RAG) — implemented in Python using TensorFlow/Keras, PyTorch, Hugging Face Transformers, and Haystack.

## Projects

| Notebook | Task | Techniques |
|---|---|---|
| [`stroke-prediction-deep-learning.ipynb`](./stroke-prediction-deep-learning.ipynb) | Predicting stroke risk from patient health records | Deep learning classification, feature engineering |
| [`handwritten-digit-classification-cnn.ipynb`](./handwritten-digit-classification-cnn.ipynb) | Classifying handwritten digits (MNIST) | Convolutional Neural Networks (CNN), Keras |
| [`sentiment-analysis-text-classification.ipynb`](./sentiment-analysis-text-classification.ipynb) | Sentiment analysis on IMDB movie reviews | Text preprocessing, text classification |
| [`deep-sentiment-analysis-embeddings.ipynb`](./deep-sentiment-analysis-embeddings.ipynb) | Sentiment analysis using learned word embeddings | Keras Embedding layer, sequence models |
| [`transformer-fine-tuning-mrpc.ipynb`](./transformer-fine-tuning-mrpc.ipynb) | Fine-tuning a transformer for paraphrase detection | Transfer learning, Hugging Face Transformers, MRPC/GLUE |
| [`rag-transfer-learning-haystack.ipynb`](./rag-transfer-learning-haystack.ipynb) | Question answering over the climate_fever dataset | Retrieval-Augmented Generation (RAG), Haystack |
| [`hate-speech-detection-arabert.ipynb`](./hate-speech-detection-arabert.ipynb) | Hate speech detection in Arabic text | AraBERT, Hugging Face, Arabic NLP |

## Stack
Python · TensorFlow/Keras · PyTorch · Hugging Face Transformers · Haystack · scikit-learn
